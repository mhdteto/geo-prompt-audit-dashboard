# Dashboard Logic

## Main views

### 1. Visibility overview

Shows the average visibility score by AI engine and week.

Recommended fields:

- Date
- Engine
- Total score
- Brand mentioned
- Position

### 2. Prompt performance

Shows which prompts generate strong, weak or missing brand mentions.

Recommended fields:

- Prompt
- Prompt category
- Engine
- Brand mentioned
- Total score

### 3. Source citation view

Tracks which sources appear in AI answers.

Recommended fields:

- Engine
- Prompt
- Cited sources
- Source type
- Citation quality

### 4. Action backlog

Converts weak scores into content or reputation actions.

Recommended fields:

- Issue
- Related prompt
- Recommended action
- Owner
- Priority

## Suggested charts

- Line chart: average score over time
- Bar chart: score by engine
- Table: low-performing prompts
- Donut chart: brand mentioned vs not mentioned
- Table: cited sources by frequency

## Operating rhythm

1. Test the same prompt set every week or month.
2. Record results in the CSV.
3. Review score movement.
4. Identify missing sources or inaccurate answers.
5. Prioritize content and reputation updates.
