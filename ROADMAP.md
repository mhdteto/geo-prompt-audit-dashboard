# Roadmap

## Current version

v1.0 - Initial public version of the GEO Prompt Audit Dashboard.

## Purpose

This project provides a lightweight structure to track how a brand appears across AI answer engines such as ChatGPT, Perplexity, Gemini, Claude and Google AI Overviews.

The goal is to help consultants, SMBs and B2B teams measure AI visibility using prompts, structured results, scoring and simple dashboard logic.

## Current scope

The current version includes:

- Sample prompt audit dataset.
- Prompt tracking structure.
- Scoring model.
- Dashboard logic.
- Example notebook.
- Documentation for running a prompt audit.
- Documentation for interpreting results.
